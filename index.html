AUTO-TUNE KEY GUIDE SYSTEM

SNAPVOX

OFFICIAL SUPPORT / OMEGA BRAIN

SNAPVOXは、ビートや楽曲のキー／スケールをもとに、 Auto-Tune系機材・プラグインで使いやすい設定キーを 素早く確認するための音楽制作サポートアプリです。

HOW TO USE

01 KEY CHECK

楽曲やビートのキー／スケールを確認します。

02 SELECT

SNAPVOXでキーを選択、または手入力します。

03 SETUP

表示されたAuto-Tune用設定を機材へ入力します。

04 ADJUST

実際の歌声やラップに合わせて微調整してください。

FEATURES

FAST GUIDE

楽曲キーから設定候補をすぐ確認。

FAQ

いいえ。ログイン不要です。
